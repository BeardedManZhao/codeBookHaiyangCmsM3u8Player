# 码本 · 海洋CMS M3U8 视频播放器

## 介绍

我们开发了一款专为 海洋CMS 定制的 M3U8视频播放器，并配套提供了完整配置教程，助您快速实现视频播放功能！

## 🔧 如何部署？

操作简单，轻松接入：

### 海洋CMS 接入

将项目中 js/player 文件夹下的所有文件上传至您的 CMS 服务器对应路径：js/player 目录；

### 其它CMS接入

我们允许所有以 `&url=` 结尾的系统来接入我们的系统！但需要根据您的系统来手动接一下，可以联系我们！
